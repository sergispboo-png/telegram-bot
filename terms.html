📋 1. Общие положения

Настоящее Пользовательское соглашение определяет условия использования Telegram-бота LuxRender.

Пользователь — любое лицо, взаимодействующее с ботом LuxRender через Telegram.

Используя сервис, вы подтверждаете согласие с условиями данного соглашения.

🤖 2. Описание сервиса

  • генерация изображений по тексту
  • редактирование изображений
  • просмотр истории генераций
  • управление балансом

Сервис использует AI-модели:

  • Google Gemini 2.5 Flash Image
  • ByteDance SeeDream 4.0

🔐 3. Регистрация и данные

  • Telegram ID и username
  • текстовые промпты
  • изображения (до 24 часов)
  • баланс и платежные операции
  • история генераций

⚖️ 4. Права и обязанности

Пользователь имеет право:

  • использовать сервис
  • просматривать историю генераций
  • пополнять баланс
  • запросить удаление данных

Пользователь обязуется:

  • не нарушать правила Telegram
  • не создавать запрещённый контент
  • не пытаться взломать сервис
  • не использовать бота для спама

💳 5. Платные функции

  • пополнение через ЮKassa
  • оплата генераций изображений
  • возврат возможен только для неиспользованных средств

⚠️ 6. Ответственность

Сервис предоставляется «как есть».

  • возможны ошибки и сбои
  • результат генерации может отличаться от ожиданий
  • сервис не отвечает за сбои сторонних API

🔞 7. Ограничения

  • использование сервиса разрешено с 16 лет
  • NSFW контент запрещён
